nMudOS is next MudOS

nMudOS is a modern mud game drive that is not only compatible with existing MudOS and FluffOS, but also creatively adds more features, while also allowing developers/game designers to quickly make mud games using LUA and JavaScript languages.

Download

Feature

Open Source

Relaxed open source protocols allow you to customize your requirements.

Compatible MudOS & FluffOS

You can run MudOS’s games and FluffOS’s games.

Multiple Programming Languages

You can use lpc Lua and JavaScript to develop your own games.

Cross-Platform

MacOS Linux and Windows are supported, and clients can also run in browsers.

A New Smart Data Sheet

Excel tables are supported, and numerical calculations are supported.

Built-in Game Engine

The built-in game engine helps you develop your game quickly.
